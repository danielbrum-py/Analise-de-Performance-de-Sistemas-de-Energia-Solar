# 📊 Solar Energy Generation Analysis

---

## 🧾 About the Project

This project aims to analyze photovoltaic solar energy generation data, exploring production patterns, temporal behavior, and system performance over time.

The main goal is to transform raw data into valuable insights for analysis and decision-making, using Python as the primary tool.

---

## 🎯 Analysis Objectives

- Understand energy generation behavior over time
- Identify daily and hourly production patterns
- Evaluate panel/system performance
- Create KPIs for energy generation monitoring
- Generate insights through exploratory data analysis

---

## 🛠️ Technologies Used

- Python 🐍
- Pandas
- Matplotlib

---

## 🧱 Project Stages

### 1. 📥 Data Import
- Reading `.csv` files using Pandas

### 2. 🧹 Data Cleaning
- Removing null values (when applicable)
- Adjusting data types
- Checking for inconsistencies

### 3. ⏱️ Date and Time Conversion
- Converting the date/time column to datetime format
- Extracting variables such as:
  - Day
  - Hour
  - Date

### 4. 📊 Exploratory Data Analysis
- Exploring energy generation behavior
- Identifying temporal patterns
- Comparing systems/panels

### 5. 📈 Data Visualization
- Energy generation over time
- Average generation by hour of the day
- Performance comparison between panels

### 6. 📊 KPI Creation
- Total energy generated
- Daily average generation
- Peak energy generation

## 🔍 Key Insights

- Energy generation follows a well-defined daily pattern, with production peaks occurring at specific times of the day.
- Generation behavior is directly related to solar irradiation availability.
- The analyzed systems/panels showed relatively balanced performance throughout the studied period.
- Daily variations indicate the influence of external factors such as weather conditions and solar intensity.
